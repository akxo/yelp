# yelp
yelp client framework using [yelp fusion](https://www.yelp.com/developers/documentation/v3)

* business endpoints
  * business search
  * phone search
  * transaction search
  * business details
  * business match
  * reviews
  * autocomplete

* event endpoints (beta)
  * event lookup
  * event search
  * featured event

* category endpoints (beta)
  * all categories
  * category details
